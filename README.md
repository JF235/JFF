# JF's Formatting

Formatando documentos.

## Uso

Adicionar a pasta no PATH

```bash
jff filename.md
# Outputs: filename.html in current directory
```

ou caso só exista somente um arquivo `.md` no diretório atual, basta executar

```bash
jff
```
